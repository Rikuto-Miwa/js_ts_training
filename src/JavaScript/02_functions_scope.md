# 関数とスコープ（30問）
1. 関数宣言と関数式の定義方法と、ホイスティング挙動の違いを説明してください。
2. 無名関数と名前付き関数式の使い分けを、デバッグ時の利点を含めて述べてください。
3. アロー関数と通常の関数で`this`・`arguments`・`prototype`の扱いがどう異なるか説明してください。
4. アロー関数をコンストラクタとして使えない理由を説明してください。
5. デフォルト引数が評価されるタイミングと、副作用を持つ初期値を設定したときの注意点を挙げてください。
6. 可変長引数（レストパラメータ）を使って合計値を計算するサンプルを書いてください。
7. `arguments`オブジェクトとレストパラメータの違いを説明し、どちらを優先して使うべきか述べてください。
8. スプレッド構文を使って関数に配列の要素を引数として渡すサンプルを書いてください。
9. `call`・`apply`・`bind`の違いを、`this`束縛と引数の渡し方に着目して説明してください。
10. `bind`で`this`を固定した関数をイベントハンドラに渡すサンプルを書いてください。
11. スコープチェーンとは何か、識別子解決の順序を説明してください。
12. グローバルスコープに値を書き込むことのリスクと、避けるための方法を2つ挙げてください。
13. ブロックスコープに閉じた`let`変数がループでどのように振る舞うか、`var`との違いを示してください。
14. 関数スコープ内で宣言される内部関数が外側の変数にアクセスできる理由を説明してください。
15. クロージャとは何か、自分なりの言葉で定義し、簡単なコード例を示してください。
16. クロージャがメモリリークを引き起こす可能性のある状況と、その回避策を述べてください。
17. 即時実行関数式（IIFE）の典型的な用途を2つ挙げてください。
18. 再帰関数で終端条件を設けないと何が起こるか、スタックオーバーフローの観点で説明してください。
19. 名前付き関数式を使うことで再帰呼び出しに利点がある理由を述べてください。
20. パラメータの分割代入を使ってオプションを受け取る関数の例を書いてください。
21. 関数が他の関数を返す高階関数の例を挙げ、その利点を説明してください。
22. 部分適用やカリー化の基本的な考え方を、関数合成の観点で説明してください。
23. イベントリスナーとしてメソッドを渡したときに`this`が意図せず変わる原因と対策を述べてください。
24. 非同期API（例: `setTimeout`）内でループ変数を正しく捕捉するための書き方を示してください。
25. `with`文が推奨されない理由をスコープの観点から説明してください。
26. `eval`がスコープに与える影響と、使用が避けられるべき理由を述べてください。
27. 関数宣言をブロック内に書いたときの挙動が実装依存になり得る理由を説明してください。
28. 「関数は第一級オブジェクトである」とはどういう意味か、実用的な例を挙げて説明してください。
29. 即時に一度だけ実行される初期化処理を、安全に関数で隠蔽するパターンを示してください。
30. スコープを意識したコード整理のためにモジュールやブロックを活用する具体的な方針を3つ挙げてください。
