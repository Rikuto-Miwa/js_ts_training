# 言語の基礎（30問）
1. JavaScriptのプリミティブ型をすべて列挙し、それぞれどんな特徴があるか説明してください。
2. `let`・`const`・`var`のスコープと再宣言/再代入の可否の違いをまとめてください。
3. 暗黙的グローバル変数が生まれるケースと、それを避ける方法を説明してください。
4. テンプレートリテラルで式展開を行うサンプルを記述し、従来の文字列連結との違いを述べてください。
5. テンプレートリテラルで複数行文字列を扱う利点を説明し、注意点があれば挙げてください。
6. ブロックスコープと関数スコープの違いを、具体例を用いて説明してください。
7. ホイスティングとは何か、`var`宣言と`let`/`const`宣言での挙動の違いを説明してください。
8. 一時的デッドゾーン（TDZ）が発生する理由と、コードでどう現れるかを示してください。
9. `typeof`演算子で得られる主な戻り値と、`typeof null`が返す値に注意が必要な理由を説明してください。
10. 算術演算子（`+ - * / % **`）の優先順位と結合規則を簡潔にまとめてください。
11. 単項プラス・マイナス演算子を使ったとき、オペランドにどんな変換が行われるか説明してください。
12. `+`演算子が数値加算ではなく文字列結合として働くのはどんな条件のときか、例を挙げてください。
13. 前置/後置インクリメント（`++i`と`i++`）の評価結果と、副作用の違いを説明してください。
14. 三項演算子の基本的な構文と、if文に置き換える場合の対応関係を示してください。
15. `if`と`switch`を使い分けるときの基準や、`switch`で`break`を忘れると何が起こるか説明してください。
16. ループで`break`と`continue`を使用する具体例を示し、使いどころを説明してください。
17. `for`ループと`while`ループの使い分けを、典型的な利用シナリオを挙げて説明してください。
18. ラベル付き文はどのようなときに使われるか、簡単なサンプルコードを書いてください。
19. 論理演算子`||`と`&&`の短絡評価がどのように働くか、式の評価順を含めて説明してください。
20. null合体演算子`??`の役割と、`||`との違いを具体例で説明してください。
21. オプショナルチェーン演算子`?.`が便利な場面と、`?.`を使ったときに`undefined`が返る条件を説明してください。
22. 分割代入（オブジェクト・配列）で基本的な値を取り出すサンプルを書き、デフォルト値の指定方法も示してください。
23. スプレッド構文`...`で配列やオブジェクトをコピーするときの注意点（浅いコピーになる点）を説明してください。
24. 数値の丸めで`Math.floor`・`Math.ceil`・`Math.round`の違いを、サンプルとともに説明してください。
25. 浮動小数点の誤差が起きる例（`0.1 + 0.2`など）と、回避/軽減策を挙げてください。
26. `Number()`・`String()`・`Boolean()`を使った明示的型変換の基本的な挙動を説明してください。
27. `isNaN`と`Number.isNaN`の挙動の違いを例で示してください。
28. `Object.is`が`===`と異なる結果を返すケース（`NaN`や`0`の符号）を挙げてください。
29. BigIntの基本的な使い方と、Numberと混在させた演算で起こるエラーについて説明してください。
30. コードを読みやすくするための命名やセミコロンの扱いなど、言語の基礎レベルで意識すべきスタイル上のポイントを3つ挙げてください。
