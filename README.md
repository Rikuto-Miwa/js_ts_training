# このリポジトリについて

このリポジトリは、Javascrit, Typescript の学習用リポジトリです。
それぞれの言語で抑えるべき記法やルールを、実際に手を動かしながら学ぶことができます。

## JavaScript学習内容
- 言語の基礎: 値とデータ型、let/const/var、テンプレートリテラル、演算子、if/for/while/switch。
- 関数とスコープ: 関数宣言/式/アロー関数、デフォルト引数と可変長引数、this/bind/call/apply、スコープとクロージャ。
- 型変換と比較: 暗黙/明示的な型変換、truthy/falsy、== と === の違い。
- オブジェクトとプロトタイプ: オブジェクトリテラル、プロパティ操作、プロトタイプチェーン、class と継承、super。
- コレクション: 配列メソッド(map/filter/reduce など)、Set/Map、反復処理(for...of/for...in、イテレーター/ジェネレーターの概念)。
- モジュール: ES Modules の import/export、デフォルト/名前付きエクスポート、モジュールスコープ。
- エラーハンドリング: try/catch/finally、throw、Error オブジェクト、独自エラーの作成。
- 非同期処理: タイマー、Promise の状態とチェーン、Promise.all/any/allSettled/race、async/await、イベントループとマイクロタスク。
- ブラウザ API: Fetch API、JSON 取り扱い、AbortController、Web Storage(localStorage/sessionStorage)、URL/History の基礎。
- DOM とイベント: DOM 取得と操作、クラス操作、イベント伝播/委譲、防止すべきデフォルト動作。
- デバッグと品質: console API、デバッガ、strict mode、リンター/フォーマッタの活用。

参照: MDN Web Docs JavaScript ガイド (https://developer.mozilla.org/ja/docs/Web/JavaScript)
